Philosophy of *nixIt

Part edited by joakim

The philosophy of *nixIt is simple: to technologically empower people with open source software like the Linux OS. It's a non-profit project aimed at serving a variety of clients, from seniors new to computing to at-risk youth opting for a productive distraction. Topics covered: basic computing, managing simple tasks using the command line, automating simple tasks using bash shell scripts, and advanced tasks such as- setting up a web server powering their own little web page.

*nixIt Philosophy

Part edited by Joakim

The philosophy of *nixIt is simple: to technologically empower people with open source software like the Linux OS. It's a non-profit project aimed at serving a variety of clients, from seniors new to computing to at-risk youth opting for a productive distraction. Topics covered: basic computing, managing simple tasks using the command line, automating simple tasks using bash shell scripts, and advanced tasks such as- setting up a web server powering their own little web page.

Education

Developing and implementing lesson plans based on client needs.

Learning

Teaching :
- How to install and configure Linux systems.
- Basic to advanced *nix commands and Bash shell scripting.
- How to install LAMP stack, and make simple websites in HTML5 and CSS3 powered by the LAMP stack.

Prep

RHCSA prep consultation: advising those taking the Red Hat Certified System Administrator (RHCSA) certification exam while respecting the NDA.
