    First Post

    Part edited by Loven Costa

    The philosophy of *NIXit is simple: to technologically empower people with open source software like the Linux OS. It's a non-profit project aimed at serving a variety of clients, from seniors new to computing to at-risk youth opting for a productive distraction. Topics covered: basic computing, managing simple tasks using the command line, automating simple tasks using bash shell scripts, and advanced tasks such as- setting up a web server powering their own little web page.
